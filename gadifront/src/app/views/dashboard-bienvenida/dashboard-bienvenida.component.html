¿Qué es GADI?

GADI es una plataforma para gestionar distributivos institucionales. Provee funcionalidades para administrar eficientemente las horas docentes y no docentes de cada uno de los miembros docentes del Instituto Superior Universitario Tecnológico del Azuay de una manera intuitiva y sencilla.
