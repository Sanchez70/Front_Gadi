SISTEMA GADI

-------LEVANTAMIENTO BACK-END-------

Para realizar el levantamiento de nuestro back-end, se deberá tener las siguientes herramientas:
-- Node js (última version)
-- SpringBoot (IDE eclipse o IntelligenIdea)
-- XAMPP
-- MySQL Workbench (opcional).
1. Descargaremos nuestro front-end en formato zip y descomprimimos.
2. Abrimos el back-end con uno de los IDE mencionados anteriormente.
3. Configuramos nuestro application properties para que apunte a un servidor de manera local.
4. Levantaremos XAMPP
5. Ejecutamos el back

-------LEVANTAMIENTO FRONT-END-------

Para realizar el levantamiento de nuestro front-end, se deberá tener algunas herramientas como:
-- Visual Studio
-- Angular 17
-- TypeScript 5.4.2
1. Descargaremos el front-end en formato zip y descomprimimos.
2. Abrimos el proyecto mediante Visual Studio.
3. Ejecutamos en la terminal "npm install" para poder instalar todas las dependencias del sistema.
4. Ejecutamos el sistema con "ng serve -o"
5. Entramos al link: http://localhost:4200/login
